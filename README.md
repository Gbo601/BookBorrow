# Book_Borrow
## 介绍
这是我大二的一个数据库课程设计，一个图书管理系统
## 使用方法：
  将项目导入IDEA，运行Start.java
## 功能：
#### 管理员：
用户管理，发布公告，图书管理
#### 用户：
查看个人信息，充值，修改信息，修改密码，图书查阅和借阅，和图书归还功能
